# Sistema de Gestão com Governança de TI e Controle de Mudanças

Este projeto tem como objetivo simular o uso de Gerenciamento de Configuração de Software (SCM) 
em um ambiente colaborativo.

O sistema está alinhado com:

- Governança de TI
- ESG (Environmental, Social and Governance)
- Controle de Versionamento
- Boas práticas de colaboração

Ferramentas utilizadas:
- Git
- GitHub
